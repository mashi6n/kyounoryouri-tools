# KyounoRyouri Tools
このリポジトリは、日本放送協会(NHK)のウェブサイト「きょうの料理」(URL)に掲載されているレシピをダウンロードし、レシピデータを抽出するためのツールを提供します。

## 注意事項
- このリポジトリは非公式であり、NHKとは一切関係がありません。
- NHKきょうの料理 はNHKの登録商標です。
- KyounoRyouri Tools は料理レシピデータを研究目的で利用するために作られたものです。ダウンロードしたデータの適切な利用については著作権法30条を遵守してください。
- ウェブサイトへの過度な負荷を避けるため、既存のダウンロード間隔を変更しないでください。

## 依存関係
- Python>=3.10
- pip もしくは uv

## インストール
1. リポジトリをクローンします。
   ```bash
   git clone https://github.com/yourusername/kyounoryouri-tools.git
   ```
2. 仮想環境を作成し、アクティベートします。
   ```bash
   pip install -e .
   ```
